# ST26 論壇 HTML 產生器 - 更新日誌

## [v6.5] 2026-05-11 | 終極自動化：DOM 原生攔截引擎
* **【架構重構】全域掃描與主動綁定**：徹底移除人工「喚醒標籤」。腳本改為直接掃描 Drupal 原生 Class (`.session__people` 與 `.session__subsession`)。
* **【營運優化】100% 零摩擦**：只要後台資料建置完成，前台導航按鈕即刻自動亮起並綁定平滑滾動。

## [v6.4] 2026-05-11 | CMS 結構化欄位解耦
* **【邏輯轉換】目標反向呼叫錨點**：設計 `.st26-trigger-speakers` 喚醒標籤，解決 Drupal 無法自訂 HTML ID 的限制。(註：此功能已於 v6.5 被原生掃描取代)

## [v6.3] 2026-05-11 | Excel 資料流精準重構
* **【資料修正】校準解析引擎**：深度解析 `ST26_官網連結.csv`。重新映射抓取座標：票價 (`c[8]`)、中英文連結 (`c[18]`, `c[19]`)，確保資料帶入零誤差。

## [v6.2] 2026-05-11 | 營運防呆與動態錨點開關
* **【UI 強化】新增防呆開關**：於「② 主題/大綱」介面加入「講師」與「議程」手動開關，避免產出尚未建置完成的無效死連結。

## [v6.0 - v6.1] 2026-05-11 | 雙語分流與 Excel 掛載
* **【自動化】連結掛載**：支援解析 Excel 中的報名連結。
* **【架構升級】雙語動態分流**：自動判斷語系並植入對應連結。具備 **Fallback 機制**（任一語系連結缺失時自動套用另一版作備案）。
* **【功能解鎖】全導覽錨點**：開啟「票價」、「講師」、「議程」三合一快速導航列。

## [v5.8 - v5.9] 2026-05-07 | 商業視覺與層級重構
* **【UI 優化】點擊暗示 (Affordance)**：強化主題區塊視覺，提升白底網頁的點擊引導。
* **【視覺降維】資訊淨化**：移除重複的地點模塊；弱化 Day Pass 字級，確保主票價的視覺統治力。
* **【動態渲染】空區塊抹除**：若無 Logo 資料，系統自動移除該區塊標題與間距，保持頁面精簡。

## [v5.6 - v5.7] 2026-05-07 | RWD 降維與語系隔離
* **【前端技術】無表格 (Table-less) 卡片化**：廢除 `<table>` 改用 `CSS Grid`。解決手機端（<768px）文字擠壓與破版問題。
* **【語系隔離】動態翻譯**：將專業商用術語與語系切換鎖定，確保 100% 正式的輸出。

## [v5.5 以前]
